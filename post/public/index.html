filter: blur()

blur() 将图像设置为高斯模糊, 默认值：0, 不可将值设置为 “百分比”

.box {
    width: 600px;
    height: 00px;
}
img {
    width: 100%;
    height: 100%;
    filter: blur(2px);     
}

filter: brightness()

应用线性乘法器，使图像看起来更光亮或者更暗淡。值为 0%，图像是全黑。值为 100%，图像无变化。值超过100%, 图像的亮度更高。如果没有设定值，则默认为 100%

.box {
    width: 600px;
    height: 400px;
}

img {
    width: 100%;
    height: 100%; 
    filter: brightness(50%);    
}

filter: contrast()

调整图像的对比度，值为 0%，图像将为完全黑色的图像。值为 100%，则图像保持不变。值超过100%时，会使图像的对比度比较低。如果缺少参数，则默认值为 100%

.box {
    width: 600px;
    height: 400px;
}
img {
    width: 100%;
    height: 100%;
    filter: contrast(180%)
}

filter: drop-shadow()

将阴影效果应用在图像上，投影实际上是将特定的颜色绘制输出到图像蒙版的模糊偏移版本。但不允许使用’inset’关键字

此属性类似于 [“box-shadow”]() 属性, 在有些浏览器品牌提供了硬件加速，box-shadow 可获得更好的性能优势。

参数：

<offset-x><offset-y> 设置阴影偏移量，offset-x 设定水平方向，offset-y 设定垂直方向。如果两个参数为 0，则设定在元素正后面

<blur-radius> (可选) 这是第三个 值. 值越大，越模糊，则阴影会变得更大更淡.不允许负值 若未设定，默认是0 (则阴影的边界很锐利).

<spread-radius> (可选) 默认值 0，负值阴影缩小，正值阴影扩大。 注意: Webkit, 以及一些其他浏览器 不支持第四个长度，如果加了也不会渲染。

(可选) 设定的颜色值

.box {
    width: 600px;
    height: 400px;
}

img {
    width: 100%;
    height: 100%;
    filter: drop-shadow(10px 10px 10px red);    
}

drop-shadow 与 box-shadow 的区别

drop-shadow 将DOM元素轮廓自然渲染出阴影，而box-shadow只是将DOM元素其路径再渲染阴影

main {
    width: 500px;
    height: 300px;
    display: flex;
    flex-direction: row;
    justify-content: space-between;
    align-items: center;
}
.box {
    width: 200px;
    height: 200px;
    border-radius: 50px;
    background: #fff;
    display: flex;
    justify-content: center;
    align-items: center;
}

.drop-shadow {
    filter: drop-shadow(0px 0px 10px rgba(0,0,0,.5));
}

.box-shadow {
    box-shadow: 0 0 10px rgba(0, 0, 0, .5)
}

<main>
    <div class="box drop-shadow">
        <span>filter: drop-shadow</span> 
    </div>
    <div class="box box-shadow">
        <span>box-shadow</span>
    </div>
</main>

filter: grayscale()

设置图像的灰度，值为 0% ~ 100% 之间。如果为设置值，则值默认 0

.box {
    width: 600px;
    height: 400px;
}

img {
    width: 100%;
    height: 100%;
    filter: grayscale(100%);
}

在帖子开头就是展示这种效果

filter: hue-rotate()

应用色调旋转，调整图像样本的色环角度数。如果缺少参数，则默认值为 0 deg, 最大值为 360 deg

.box {
    width: 600px;
    height: 400px;
}
img {
    width: 100%;
    height: 100%;
    filter: hue-rotate(100deg)    
}

filter: invert()

反转图像，参数值定义了转换比例。值为100%则完全反转，值为0%则保持不变。 0% ~ 100% 之间的值会以线性乘数。如果缺少参数，默认值 100%。（负数值无效）

.box {
    width: 600px;
    height: 400px;
}
img {
    width: 100%;
    height: 100%;
    filter: invert(100%)    
}

filter: opacity()

图像的透明度，值为 0% ~ 100% 之间效果是以线性乘数（值也可为0 ~ 1）。如果缺少参数，则使用100%。此属性类似于更成熟的不透明度属性，区别在于使用过滤器，有些浏览器能提供更好渲染的性能。（使用负值无效）

.box {
    width: 600px;
    height: 400px;
}
img {
    width: 100%;
    height: 100%;
    filter: opacity(.2);
    -webkit-filter: opacity(20%); 
}

saturate()

转换图像的饱和度，。值为 0% 则完全不饱和，值为 100% 使其保持不变。值允许超过 100%。如果缺少参数，则默认值 100%

.box {
    width: 600px;
    height: 400px;
}
img {
    width: 100%;
    height: 100%;
    filter: saturate(20%);
    -webkit-filter: saturate(20%); 
}

filter: sepia()

将图像转为深褐色，值为 100% 则完全是深褐色，值为 0% 则保持不变。0% ~ 100% 效果以线性乘数渲染。如果缺少参数，默认值 0

.box {
    width: 600px;
    height: 400px;
}
img {
    width: 100%;
    height: 100%;
    filter: sepia(50%);
    -webkit-filter: sepia(50%); 
}

filter: url()

该url()函数获取指定SVG滤镜的XML文件的位置，并且可以包括到特定过滤器元素的锚。

.box {
    width: 600px;
    height: 400px;
}
img {
    width: 100%;
    height: 100%;
    filter: url(#goo);
}

因为 svg 滤镜涉及 feColorMatrix 矩阵计算，帖子也足够消化，就不长篇……
